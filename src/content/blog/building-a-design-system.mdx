This is a sample post. It's written in **MDX**, so it's regular Markdown with
the option to drop in components when plain prose isn't enough.

## Why one accent color

Constraining the palette to a single teal accent (plus tints and a dark "ink"
shade for text) keeps everything cohesive. The accent shows up in links, the
primary button gradient, list markers, and the hero grid, and nowhere else
fights it for attention.

Here's an inline bit of `code`, and a block:

```ts
type Token = `--sl-${string}`;

const accent: Token = "--sl-accent"; // #5EC8C0
```
